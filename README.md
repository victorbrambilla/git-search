# GitHubRepo

<h1>Como Rodar</h1>

1- Clonar ESTE repositório para sua máquina local

2- Abrir o repositório no editor de código

3- No terminal, na pasta do projeto dê o comando
~~~Terminal
ng serve
~~~

5- Abra o seguinte link em algum navegador
~~~Terminal
http://localhost:4200
~~~
